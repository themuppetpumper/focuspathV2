📖 What is FocusPath?

FocusPath is an all-in-one study companion that works completely offline. It's designed to help you create, practice, and track your learning — all from one place. No accounts, no cloud — everything is stored right in your browser.

Here's what you get:

  • Flash Cards — Create card decks by subject and study them with timed sessions
  • Tests — Build custom multiple-choice tests and track your scores over time
  • Pomodoro Timer — Focus in 25-minute intervals with automatic break scheduling
  • Goal Tracker — Set goals with milestones and deadlines
  • Recall Journal — Write daily reflections using structured prompts
  • Study Planner — Plan your weekly study schedule with a visual timetable
  • Habit Tracker — Build and maintain daily study habits with streak tracking
  • Cornell Notes — Take structured notes using the proven Cornell method
  • Pre/Post Compare — Compare test attempts side-by-side to see your growth
  • Statistics & Achievements — Track all your progress with charts, badges, and streaks

🗂️ Flash Cards

Create flashcard sets organized by subject. Each card has a front (question/term) and back (answer/definition).

How to use
  • Go to Flash Cards and click New Set
  • Give your set a name, pick a subject, and add cards
  • Click Study on any set to start a timed session
  • Click the card to flip it and see the answer
  • Your sessions are tracked — cards seen, time spent, and subject coverage
Tips
  • Keep cards short — one concept per card works best
  • Study across multiple subjects to keep things balanced
  • Revisit sets you haven't touched in a while for spaced repetition
Importing & Exporting Sets
  • Click Export Sets in the sidebar to export your flashcard data as a JSON file
  • If you have a set selected, you'll be asked whether to export just that set or all sets
  • You can optionally encrypt the export with a password for secure sharing
  • Click Import Sets and select a .json file to import flashcard sets
  • Encrypted files will prompt you for the password before importing
  • Duplicate set IDs are handled automatically — imported sets get new IDs if needed
  • Only flashcard files are accepted — test files will be rejected with a helpful message

📝 Tests

Build multiple-choice tests with as many questions as you like, then take them to measure your knowledge.

How to use
  • Go to Tests and click New Test
  • Add a test name, subject, and write your questions with answer options
  • Mark the correct answer for each question
  • Click Take Test to attempt it — your score and time are recorded
  • Review your answers after completing the test
Tips
  • Take the same test multiple times to track improvement
  • Use the Pre/Post Compare tool to see exactly which questions you improved on
  • The Statistics page shows your score trends over time by subject
Importing & Exporting Tests
  • Click Export Tests in the sidebar to export your test data as a JSON file
  • If you have a test selected, you'll be asked whether to export just that test or all tests
  • You can optionally encrypt the export with a password — great for teachers sharing tests with students
  • Password-protected tests are automatically locked when imported, preventing students from editing them
  • Click Import Tests and select a .json file to import tests
  • Encrypted files will prompt for the password — incorrect passwords cause that test to be skipped
  • Duplicate test IDs are handled automatically — imported tests get new IDs if needed
  • Only test files are accepted — flashcard files will be rejected with a helpful message

🍅 Pomodoro Timer

The Pomodoro Technique breaks work into focused intervals (25 min) separated by short breaks (5 min), with a longer break (15 min) every 4 sessions.

How to use
  • Go to Pomodoro Timer and click Start to begin a 25-minute focus session
  • When the timer ends, a break starts automatically
  • You can also switch to a break mid-session — your focus progress is saved and resumes after the break
  • A floating mini-timer appears on all other pages so you can track time while studying
  • Hover over the mini-timer for controls: take a break, resume focus, or stop
Tips
  • Commit to finishing a full focus session before checking other tasks
  • Use breaks to stretch, hydrate, or look away from the screen
  • Your daily sessions and focus time are tracked on the Statistics page

🎯 Goal Tracker

Set study goals with optional deadlines and break them into milestones to track progress step by step.

How to use
  • Click New Goal and fill in the title, category, and optional deadline
  • Add milestones — these are checkable sub-tasks that make up your goal
  • Check off milestones as you complete them — the progress bar updates automatically
  • Mark the entire goal as complete when you're done

🧠 Recall Journal

Write daily reflections about what you've learned. Recalling information from memory is one of the most effective study techniques.

How to use
  • Go to Recall Journal and click New Entry
  • Choose a prompt (or write freely) and recall what you studied today
  • Entries are grouped by date — you can search and filter them
  • Your writing streak is tracked on the Statistics page
Why it works

Active recall — retrieving information from memory — strengthens neural pathways far more effectively than re-reading notes. Writing without looking at your materials forces your brain to work harder, which means better retention.

📅 Study Planner

Create a visual weekly timetable with color-coded study blocks for each subject.

How to use
  • Click Add Block to create a study block
  • Set the subject, day of the week, start/end times, and pick a color
  • Blocks appear on a weekly grid — drag-free visual planning
  • Click any block to edit or delete it

✅ Habit Tracker

Define daily, weekday, or weekend habits and check them off each day. Build streaks and see your consistency on a heatmap.

How to use
  • Click New Habit — name it, pick an icon, color, and frequency
  • Each day, check the box next to habits you've completed
  • Your current streak and a 16-week heatmap show your consistency
  • Streaks and completion rates are tracked in Statistics
Tips
  • Start with 2–3 habits — you can always add more later
  • Don't break the chain! Consecutive days build strong habits

📝 Cornell Notes

The Cornell method divides your notes into three sections: Cues (key questions/terms), Notes (detailed notes), and Summary (a brief recap).

How to use
  • Click New Note and give it a title
  • Fill in the Cue column with key terms or questions
  • Write detailed Notes in the main section
  • Write a Summary at the bottom in your own words
  • Notes auto-save as you type

📊 Pre/Post Compare

Compare two attempts of the same test side-by-side to see exactly where you improved and what still needs work.

How to use
  • Go to Pre/Post Compare
  • Select a test that you've taken at least twice
  • The tool shows a timeline of all attempts plus a detailed question-by-question comparison
  • Green = improved, Red = regressed, Gray = stayed the same

📈 Statistics & Achievements

The Statistics page pulls together data from all your study activities into one dashboard.

What you'll find
  • Stats Home — Today's Focus card showing daily Pomodoro/habits/journal/goals activity, plus all-time trophy cards
  • Test Stats — Score trends, subject breakdown, time analysis, and suggestions
  • Flashcard Stats — Study depth analysis, subject correlation with test performance, and session trends
  • Tools Stats — Pomodoro sessions over time, habit completion rate, journal entry frequency, and goal progress
  • Trophy Room — 60+ unlockable badges across all study activities

📦 Import & Export

FocusPath lets you export and import your tests and flashcard sets as JSON files. This is useful for backups, sharing with classmates, or transferring data between devices.

How exporting works
  • On the Tests or Flashcards page, click the Export button in the sidebar
  • If you have an item selected, you'll be asked: export just that item, or everything?
  • Choose a filename and optionally set a password to encrypt the file
  • The file downloads as .json — share it, email it, or keep it as a backup
How importing works
  • Click the Import button in the sidebar and pick a .json file
  • If the file is encrypted, you'll be prompted for the password
  • Imported items are added alongside your existing data — nothing is overwritten
  • If an imported item has the same ID as an existing one, it gets a fresh ID automatically
File format safety
  • FocusPath marks exported files with a type identifier so they can't be mixed up
  • Importing a test file on the Flashcards page (or vice versa) will show a clear error message directing you to the correct page
  • Corrupted or invalid JSON files are rejected with an error message
Password-protected exports
  • Both tests and flashcards support AES-256 encrypted exports
  • Encryption uses PBKDF2 key derivation — your password is never stored in the file
  • Password-protected tests are automatically locked on import, preventing editing — ideal for teachers distributing exams
  • If the wrong password is entered during import, that item is skipped (not lost)
Tips for sharing
  • Teachers: Export a test with a password, then share the file and password separately with students
  • Study groups: Export flashcard sets and share the JSON file — everyone can import it
  • Backups: Periodically export all tests and all flashcard sets to keep a backup copy
  • New device: Export everything from one browser and import on another to transfer your data

💡 Study Tips

  • Active Recall — Test yourself instead of re-reading. Use flash cards and the Recall Journal.
  • Spaced Repetition — Review material at increasing intervals. Don't cram everything in one day.
  • Pomodoro Sessions — 25 minutes of focused work beats hours of distracted studying.
  • Track Everything — The Statistics page shows patterns you'd never notice on your own.
  • Be Consistent — Small daily effort beats occasional marathon sessions. Use the Habit Tracker to stay on course.
  • Mix Subjects — Interleaving different topics in one session improves long-term retention.
  • Write Summaries — After each study session, write a quick journal entry about what you learned without looking at your notes.

❓ Frequently Asked Questions

Where is my data stored?

Everything is stored in your browser's local storage. No data is sent anywhere — it all stays on your device. This means it works fully offline but is tied to this browser.

Can I use this offline?

Yes! FocusPath is a Progressive Web App (PWA). After your first visit, all pages and assets are cached. You can study without an internet connection.

How do I install it as an app?

On Chrome/Edge, look for the install icon in the address bar. On mobile, tap "Add to Home Screen" from the browser menu. It will work just like a native app.

What does "Soft Reset" do?

Soft reset hides existing items (tests or flashcard sets) from the main page but keeps them in your statistics. It's useful for a fresh start without losing historical data.

What does "Hard Reset" do?

Hard reset erases all tests, attempts, flashcard sets, and sessions — but preserves your achievement badges. It cannot be undone.

What's the 🧰 button in the nav bar?

That's the Tools Drawer. It gives you quick access to all study tools from any page — Pomodoro, Goals, Journal, Planner, Habits, Cornell Notes, and Compare.

How do themes work?

Click the palette icon 🎨 on the home page to open the theme picker. Themes change the app's colors across all pages. Your choice is saved automatically.

Can I see this welcome guide again?

You can always come back to this Help page! It's accessible from the 🧰 Tools menu on every page.
